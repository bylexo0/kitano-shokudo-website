# Kitano Shokudo (Otto Bistro) Website

A modern, bilingual (English/French) website for Kitano Shokudo, a Japanese bistro in Montreal.

## Features

- **Bilingual Support**: Toggle between English and French.
- **Mobile Responsive**: Optimized for all devices.
- **Menu System**: Interactive lunch and dinner menus.
- **Reservations**: Integrated with OpenTable.
- **Social Media**: Links to Instagram and Facebook.
- **Location**: Google Maps integration.

## Technologies

- HTML5
- CSS3 (Custom Design System)
- Vanilla JavaScript
